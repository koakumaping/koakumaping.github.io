起因

昨天申请了了http://nag24.org的域名，google到了一篇文章GitHub Pages 绑定来自阿里云的域名，按照他的方法试了很久，一直404

我的方法

其实很简单的

  1. 添加CNAME文件到根目录,里面写nag24.org
  2. 在阿里云域名控制台中的解析设置中添加一条CNAME记录，如下图所示
  3. 等待生效
  4. 大概就几分钟吧，访问http://nag24.org就能正确显示了

注意事项

  1. CNAME文件必须添加，里面的域名不能带有http://前缀，不然即使你域名解析设置正确了访问你自己的域名Github Pages会提示404
  2. 用CNAME解析就足够了。如果想要解析成类似http://blog.nag24.org你可以把主机记录中的@改成blog
